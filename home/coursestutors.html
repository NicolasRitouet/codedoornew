The courses we offer

All courses of our programming school are divided into modules. Depending on the area of interest, you can support the students in the following areas.

Introduction to Coding

The students will be able to learn the basics of coding with a focus on web development: HTML, CSS, JavaScript, Git, HTTP, SQL.

Object Oriented Programming

Companies have told us that they would love to have a course that will empower them to build what they want. Dive with the students into Python and Javascript and its frameworks.

IT Security

Want to teach about cyber security? We are an official Cisco Network Academy. From basic courses to security experts, help the students to improve their security skills.

Machine Learning

Become a machine learning engineer tutor and help the students to apply predictive models to massive data sets in fields like education, finance, healthcare or robotics.

Android Developer

Did you know that about 80% of all smart phones run on Android? Of course you do, if you are an Android Developer. Empower the students to enter your world and to build Android Apps.

iOS Developer

Did you know that there are only 13,000 iOS developers in Germany? Help us to fill that gap and build amazing iOS apps and support future co-workers.
